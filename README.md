# **Resonance Assistant**

</div>

ResonanceAssistant是一个基于[MAAFramework](https://github.com/MaaXYZ/MaaFramework)框架实现游戏[~~唢呐黑丝~~《雷索纳斯》](https://soli-reso.com/)自动清日常和~~最大化列车跑商收益~~的软件。

基于全新架构的 列车长 小助手。图像技术 + 模拟控制，解放双手！
由 MaaFramework 强力驱动！

什么你问我为好多功能还没实现？~~因为咕咕咕了还没做完~~

已经在做啦！
